# Openwalls

![Banner](assets/openwalls_banner.svg)

Openwalls is a powerful, lightweight, and intelligent desktop wallpaper engine designed for Windows. It supports video, image, and solid color wallpapers with a modern, macOS-inspired dashboard.

## Features

- Dynamic Wallpapers: Support for video loops and high-quality images.
- Smart Pause Optimization: Automatically pauses video playback when windows cover the desktop to save CPU and GPU resources.
- Window 11 Ready: Advanced detection for snapped and maximized windows.
- Custom Library: Easily create and manage wallpaper presets.
- Minimal Footprint: Optimized background performance.

## Getting Started

1. Download the latest release.
2. Launch the application.
3. Use the Tray Icon in the taskbar to open the Settings dashboard.
4. Select or add your favorite wallpaper to the library.

## Compatible On

- Windows 11
- Windows 10
- Windows Server 2019+
